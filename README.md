# Gauge tests for Gauge LSP features
[Subcutaneous](https://www.martinfowler.com/bliki/SubcutaneousTest.html) tests for the lsp feature of Gauge.

# Pre-requisites
* Node-js
* [Gauge](https://docs.getgauge.io/installing.html)
* Install Gauge-js
```
gauge install js
```

# Execution
* npm install
* npm test
